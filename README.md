# quoridor

> An abstract strategy game.

## Why?

 - Fun and easy to play.
 - Highly expressive API.
 - Modular design.

## Install

````sh
npm install quoridor --save
````

## Usage

Get it into your program.
```js
const quoridor = require('quoridor');
```
